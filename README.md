## About The Project

[Project Country App](https://serghino.github.io/Country-App/)

A simple app to search for countries by name, capital, region and more.

Here's why:
* Improve your search experience by using the [Country API](https://restcountries.com/).
* Create a simples components without any dependencies to keep the app simple.
* Practicing good practices with the last feautures of the [Angular](https://angular.io/) framework.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/serghino/Country-App.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Init project
   ```sh
   npm run start
   ```
